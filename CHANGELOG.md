# Changelog

## 1.7.0 — Emerald UI, dark mode, and smooth controls

- Replaced the pink/blue palette with the EverythingUTM emerald palette.
- Added a persistent dark mode with a circular reveal animation.
- Added GDI+ anti-aliasing for rounded cards, switches, slider, and knob.
- Removed the in-window equalizer visualizer from the status card.
- Slider dragging no longer writes the APO config on every mouse event; it commits once on release.
- Increased animation refresh from 12.5 FPS to 60 FPS.
- Added smooth switch-thumb interpolation.
- Exiting now writes 100% / 0.00 dB before the process terminates.
- Updated the application and tray icon to an emerald gradient.
